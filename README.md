# SQL-Data-Analytics-Project
An end-to-end SQL data analytics project demonstrating database exploration, data cleaning, SQL querying, and business insights generation.

## Project Objectives

- Explore and analyze bookstore database tables.
- Perform data analysis using SQL queries.
- Understand customer purchasing behavior.
- Analyze book sales and inventory performance.
- Generate business insights using SQL.

#  Tools & Technologies Used

- **PostgreSQL**

# SQL Analysis Performed

## Basic Analysis

The project includes SQL queries to:

- Retrieve books by genre.
- Find books published after a specific year.
- Analyze customers by country.
- Filter orders based on date.
- Calculate total available stock.
- Identify the most expensive and lowest-stock books.
- Calculate total revenue generated.


## Advanced Analysis

The project includes:

- Genre-wise book sales analysis.
- Average price analysis by genre.
- Customers with multiple orders.
- Most frequently ordered books.
- Top expensive books by category.
- Author-wise sales performance.
- High-spending customer analysis.
- Remaining inventory calculation after sales.
